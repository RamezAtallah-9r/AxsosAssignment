# 🥷 Dojo Ninja Shell

A Django-based web application that manages Dojos and Ninjas.  
Each Ninja belongs to a specific Dojo, and the system allows creating, viewing, and deleting Dojos easily.

---

## 📌 Project Description

The **Dojo Ninja Shell** project is a simple relational management system built using Django.  
It demonstrates how to work with One-to-Many relationships (Dojo → Ninjas), form handling, and database operations such as create and delete.

---

## 🚀 Features

- Create new Dojos (name, city, state)
- Add Ninjas and assign them to a Dojo
- View all Dojos and their related Ninjas
- Delete Dojos
- Simple and clean UI for data management
- Demonstrates Django ORM relationships

---

## 🧰 Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, tailwind css (Django Templates)
- Database: SQLite (default Django DB)
- ORM: Django ORM
